
# ZAMSX TOOLS

Dokumentasi untuk menggunakan TOOLS ZAMSX

Pastikan Python sudah terinstall


## Dokumentasi

Clone Repository

```bash
git clone https://github.com/zams-xit/tools
```

Pindah ke direktori Tools 

```bash
cd tools
```

Instal Module

```bash
python install.py
```

Run Tools Script

```bash
python tools-zams.py
```
